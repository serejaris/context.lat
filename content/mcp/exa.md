## Что это

Exa MCP сервер даёт AI возможность искать информацию в интернете. В отличие от обычного поиска, Exa оптимизирован для AI — возвращает чистый контент, готовый к обработке.

## Возможности

- Семантический веб-поиск
- Поиск кода и технической документации
- Исследование компаний
- Результаты очищены от рекламы и мусора

## Как подключить

```json
{
  "mcpServers": {
    "exa": {
      "command": "npx",
      "args": ["-y", "exa-mcp-server"],
      "env": {
        "EXA_API_KEY": "your-api-key"
      }
    }
  }
}
```
