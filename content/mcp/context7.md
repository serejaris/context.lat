## Что это

Context7 [MCP сервер](/concepts/mcp/) подключает свежую документацию библиотек и фреймворков к AI-агенту. Решает проблему среза знаний: модель обучена на данных до определённой даты и может не знать новые API. Context7 возвращает актуальные сигнатуры функций и примеры.

## Возможности

- Свежая документация популярных библиотек и фреймворков
- Автоматическое обновление при появлении новых версий
- Корректные сигнатуры функций и примеры кода
- Снижение риска работы по устаревшим знаниям модели

## Как подключить

```bash
claude mcp add context7 -- npx -y @upstash/context7-mcp@latest
```

## Когда использовать

Когда агент работает с библиотеками, у которых часто меняется API. Context7 дополняет Exa: Exa ищет по всему интернету, Context7 специализируется на документации.
