## Что это

Контекст — это вся информация, которую AI видит при обработке запроса. Управление контекстом — ключевой навык для работы с большими проектами.

## Инструменты управления контекстом

### CLAUDE.md
Файл в корне проекта, который Claude Code читает автоматически. Содержит правила проекта, стек, паттерны.

### .cursor/rules
Аналог CLAUDE.md для Cursor. Набор правил, которые IDE подгружает автоматически.

### Правила проекта
- Описание архитектуры
- Стандарты кодирования
- Частые ошибки и как их избегать

## Стратегии

### 1. Иерархия контекста
Глобальные правила → правила проекта → правила файла → промпт

### 2. Не перегружай
Контекстное окно ограничено. Давай только релевантную информацию.

### 3. Структурируй
Используй заголовки, списки, чёткую структуру. AI лучше понимает структурированный текст.

### 4. Обновляй
Правила устаревают. Регулярно ревьюй и обновляй CLAUDE.md и rules.
