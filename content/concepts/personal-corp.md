## Определение

Личная корпорация (Personal Corporation) — концепция организации работы, при которой один человек выступает CEO, а роли отделов (редакция, исследования, разработка, маркетинг) выполняют AI-агенты. Каждый агент получает системный промпт с описанием своей роли, инструменты и задачи через GitHub Issues.

Сэм Альтман прогнозировал первую компанию-единорог с одним сотрудником. Дарио Амодеи оценил вероятность этого в 2026 году на уровне 70-80%.

## Архитектура

```
CEO (человек)
    │
    ├── Редакция (агент) — контент, публикации
    ├── Исследования (агент) — анализ, ресёрч
    ├── Разработка (агент) — код, деплой
    ├── Маркетинг (агент) — воронки, рассылки
    └── Стратегия (агент) — планирование, OKR
```

Каждый "отдел":
- Получает системный промпт с описанием роли
- Имеет доступ к нужным инструментам
- Работает через GitHub Issues как общую память
- Использует скиллы как стандартные процедуры

## Инструменты

- **GitHub Issues** — задачи и память между сессиями
- **Skills** — SOP для каждого отдела
- **Субагенты** — параллельная работа нескольких агентов
- **CLAUDE.md** — документация и правила проекта
