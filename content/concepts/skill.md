## Определение

Скиллы — переиспользуемые промпты в формате файлов `SKILL.md`, хранящиеся в папке `.claude/skills/`. Это программирование агента на естественном языке: один раз описываешь процедуру — агент выполняет её по команде. Аналог SOP (Standard Operating Procedure) для AI-агентов.

Скиллы — «новый MVP»: полноценная программа без написания кода.

## Быстрый старт

### Создать скилл

Создать файл `.claude/skills/publish-post/SKILL.md`:

```markdown
# Публикация поста в Telegram

## Шаги
1. Прочитай черновик из drafts/
2. Проверь орфографию
3. Сгенерируй превью-картинку
4. Опубликуй через Telegram Bot API
5. Архивируй черновик в published/
```

### Вызвать скилл

В Claude Code: `/publish-post` — агент выполнит все шаги по инструкции.

### Установить готовый скилл

```bash
# Superpowers — рекомендуемая библиотека скиллов
claude install-skill https://github.com/obra/superpowers
```

## Два уровня

| Уровень | Путь | Область |
|---------|------|---------|
| **Глобальные** | `~/.claude/skills/` | Все проекты |
| **Локальные** | `.claude/skills/` в проекте | Конкретный проект |

## Аналоги в других инструментах

| Инструмент | Название | Путь |
|------------|----------|------|
| Claude Code | Skills | `.claude/skills/<name>/SKILL.md` |
| Cursor | Rules | `.cursor/rules/*.md` |
| Windsurf | Rules + Skills | `.windsurfrules`, `.windsurf/skills/` |
| VS Code Copilot | Prompt Files | `.github/prompts/*.prompt.md` |

## Скиллы vs промпты

| | Промпт | Скилл |
|---|---|---|
| Хранение | В голове / clipboard | В файле проекта |
| Переиспользование | Копирование | Команда `/skill-name` |
| Контекст | Нужно каждый раз задавать | Встроен в файл |
| Версионирование | Нет | Через git |
| Масштаб | Одна задача | Полная процедура |

## Примеры скиллов

- **Публикация поста** в Telegram-канал: сбор → редактура → публикация
- **Code review** по чеклисту: безопасность, типы, тесты
- **Деплой с проверками**: билд → тесты → деплой → health check
- **Brainstorming** (Superpowers): интервью → варианты → дизайн-документ → план реализации
