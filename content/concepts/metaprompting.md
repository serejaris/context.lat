## Определение

Метапромтинг — подход, при котором модель получает инструкцию о том, как обрабатывать задачу, а не просьбу дать готовый ответ. Вместо «сделай мне сайт» — «задай мне пять уточняющих вопросов, и в конце сформулируй промпт на создание сайта».

Это особенно полезно для людей без технического бэкграунда — AI помогает структурировать задачу до того, как начнётся реализация. Метапромпт вытаскивает из головы то, что сложно сформулировать самостоятельно.

## Как работает

Обычный промпт: ты формулируешь задачу → AI выполняет.
Метапромпт: ты описываешь направление → AI задаёт вопросы → вы вместе формулируете точную задачу.

```
Ты                          AI
─────                       ─────
"Хочу трекер подписок"  →   "Вопрос 1: Какие подписки?
                             a) Стриминг  b) SaaS  c) Все"
"b) SaaS"               →   "Вопрос 2: Нужна авторизация?
                             a) Да  b) Нет  c) Позже"
"a) Да"                  →   ...
                         →   "Вот промпт для создания
                              приложения: ..."
```

## Шаблон

> Я хочу сделать [описание]. Задай мне пять вопросов с тремя вариантами ответов, по одному. С каждым вопросом давай комментарии эксперта. В конце дай мне промпт на создание приложения без указания технических деталей.

## Варианты применения

- **Структурирование идеи** — метапромпт как первый шаг перед [PRD](/concepts/prd/). Результат: структурированное описание того, что строим
- **Определение темы** — «Задавай 5 вопросов с тремя вариантами ответов, по одному... в конце дай структурированный ответ»
- **Оркестрация** — сильная модель генерирует промпты для более слабых моделей
- **PRD через метапромпт** — на Уроке 2 студенты генерируют PRD для Telegram-бота через цепочку уточняющих вопросов
- **Три промпта от идеи до roadmap** — метапромптинг → декомпозиция → roadmap с критериями проверки

## Примеры из курса

### Урок 1: трекер подписок

```
Я хочу сделать трекер подписок. Задай мне пять вопросов
с тремя вариантами ответов по одному. С каждым вопросом
давай комментарии эксперта. В конце дай мне промпт на
создание приложения без указания технических деталей.
```

Результат: за 3-5 минут диалога студент получает точный промпт, который генерирует рабочее приложение в [v0](/tools/v0/).

### Урок 2: тема Telegram-канала

```
Задавай 5 вопросов с тремя вариантами ответов, по одному.
Тема: определить направление Telegram-канала.
В конце дай структурированный ответ с нишей, темами
и контент-планом.
```

## Когда НЕ нужен

- Задача уже чётко сформулирована — метапромпт добавит лишний шаг
- Быстрые правки: «измени цвет кнопки на синий» — не нужен диалог
- Технические задачи с конкретным стеком и спекой — лучше дать прямой промпт
