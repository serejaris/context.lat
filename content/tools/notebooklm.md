## Обзор

NotebookLM — AI-инструмент от Google для работы с загруженными материалами. Отличается от обычных чат-ботов: отвечает только на основе ваших источников, цитируя конкретные места. Не выдумывает — если ответа нет в источниках, скажет об этом.

## Быстрый старт

1. Открыть [notebooklm.google.com](https://notebooklm.google.com)
2. Создать notebook → загрузить источники (ссылки, PDF, текст)
3. Задать вопрос в чате — NotebookLM ответит только из загруженных материалов
4. Каждый ответ содержит ссылки на конкретные места в источниках

## Примеры использования

**Справочник по пройденным урокам:**
```
Загрузить: конспект Урока 1, конспект Урока 2
Вопрос: "Как работает метапромптинг?"
→ Ответ с цитатами из конкретных уроков
```

**Анализ документации библиотеки:**
```
Загрузить: ссылку на docs.supabase.com
Вопрос: "Как настроить Row Level Security для таблицы users?"
→ Пошаговая инструкция из официальных доков
```

**Audio Overview (подкаст):**

NotebookLM генерирует аудио-обсуждение из загруженных материалов — два AI-ведущих обсуждают тему в формате подкаста. Полезно для повторения материала на ходу.

## NotebookLM vs обычные чат-боты

| Критерий | NotebookLM | [Claude](/models/claude/) / [ChatGPT](/tools/chatgpt/) |
|----------|------------|-----|
| Источник ответов | Только загруженные материалы | Всё, на чём обучена модель |
| Галлюцинации | Минимум (ограничен источниками) | Возможны |
| Цитирование | Точные ссылки на места в документах | Нет |
| Генерация нового | Нет (только анализ) | Да |

## Когда использовать

- **Повторение материала** — загрузить конспекты, задавать вопросы
- **Подготовка к менторингу** — загрузить все артефакты менти, получить контекст
- **Работа с документацией** — загрузить доки, получить ответы с цитатами
- **Подкасты** — Audio Overview для повторения на ходу
