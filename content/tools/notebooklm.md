## Обзор

NotebookLM — AI-инструмент от Google для работы с загруженными материалами. Отличается от обычных чат-ботов: отвечает только на основе ваших источников, цитируя конкретные места. Не выдумывает — если ответа нет в источниках, скажет об этом.

## Как использовать

1. Создать notebook на [notebooklm.google.com](https://notebooklm.google.com)
2. Загрузить источники: ссылки, PDF, текст
3. Задать вопросы в чате — NotebookLM ответит только из загруженных материалов

## Для чего

- Справочник по пройденному материалу — загрузите конспекты, задавайте вопросы
- Анализ документации — загрузите доки библиотеки, разберитесь в API
- Генерация подкастов — NotebookLM создаёт аудио-обсуждение из ваших материалов

## Для кого

Для всех, кому нужно быстро разобраться в объёмном материале с гарантией, что ответы основаны на конкретных источниках.
