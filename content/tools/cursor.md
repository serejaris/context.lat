## Обзор

Cursor — самый популярный AI-инструмент для кодинга. Это форк VS Code с встроенным AI, который понимает ваш проект и помогает писать код быстрее.

## Ключевые фишки

- **Composer**: создание и редактирование нескольких файлов одновременно по текстовому описанию
- **Tab**: умное автодополнение, которое предсказывает следующий код
- **Chat**: обсуждение кода прямо в контексте файла
- **Background Agent**: автономное выполнение задач в фоне

## Модели

Cursor поддерживает Claude, GPT, Gemini и другие модели. Можно переключаться между ними в зависимости от задачи.

## Для кого

Идеален для тех, кто уже использует VS Code. Низкий порог входа, бесплатный тариф.
