## Обзор

Bolt.new — AI-билдер от StackBlitz для генерации полноценных веб-приложений прямо в браузере. Описываете что хотите — получаете работающий проект с фронтендом и бэкендом. Ничего не нужно устанавливать — всё работает в WebContainer прямо в браузере.

## Быстрый старт

1. Открыть [bolt.new](https://bolt.new)
2. Описать приложение:

```
Создай дашборд для отслеживания привычек.
Календарь с отметками, статистика за неделю/месяц,
графики прогресса. Минималистичный дизайн.
```

3. Bolt генерирует код в реальном времени — видно как создаётся проект
4. Итерировать через чат: «Добавь тёмную тему», «Измени цвет графиков»
5. Деплой: кнопка Deploy → Netlify → публичная ссылка

## Ключевые возможности

- **Full-stack**: генерирует не только UI, но и бэкенд-логику
- **Мульти-стек**: поддерживает React, Next.js, Vite, Node.js и другие
- **WebContainer**: код выполняется прямо в браузере, без серверов
- **Интеграция с [Supabase](/tools/supabase/)**: база данных и авторизация
- **Git-экспорт**: можно выгрузить код в GitHub для доработки в [Cursor](/tools/cursor/)

## Сравнение с альтернативами

| Критерий | Bolt.new | [v0](/tools/v0/) | [Lovable](/tools/lovable/) |
|----------|---------|----|---------| 
| Стек | Любой (Stackblitz) | React + Tailwind | React + Supabase |
| Деплой | Netlify | Vercel | Собственный |
| Фокус | Full-stack проекты | UI-компоненты | Продуктовые MVP |
| Бэкенд | Да | Через Supabase | Через Supabase |
| Сила | Мульти-стек, полный проект | Красивый UI | Визуальный редактор |

## Когда выбрать Bolt

- Нужен полноценный full-stack проект (не только UI)
- Хотите работать со стеком, отличным от React
- Не хотите устанавливать ничего локально
