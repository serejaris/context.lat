## Обзор

Gemini — AI-ассистент и семейство моделей от Google. Выделяется визуально богатыми интерфейсами по умолчанию и самым большим контекстным окном — до 2M [токенов](/concepts/token/) у Gemini 2.5 Pro. Для работы с кодом слабее [Claude Code](/tools/claude-code/) и [Cursor](/tools/cursor/), но хорош для генерации контента и работы с большими документами.

## Модели

| Модель | Контекст | Цена API | Сила |
|--------|----------|----------|------|
| **Gemini 2.5 Pro** | 2M | $1.25/1M input | Кодинг (SWE-bench 77.4%), длинный контекст |
| **Gemini 2.5 Flash** | 1M | $0.15/1M input | Самая дешёвая модель для массовых задач |

Бесплатный доступ через [AI Studio](https://aistudio.google.com).

## Быстрый старт

1. Открыть [gemini.google.com](https://gemini.google.com)
2. Ввести метапромпт для генерации сайта — Gemini создаст визуально богатый результат
3. Для анализа текстов: загрузить экспорт Telegram-канала → попросить извлечь стиль и тон

## Сильные стороны

- **Визуальные артефакты** — лучший фронтенд по умолчанию при генерации сайтов
- **Анализ стиля** — хорошо работает с экспортами текстов для извлечения тона и голоса
- **Работа с документами** — контекст до 2M позволяет обрабатывать целые книги
- **Генерация контента** — описан студентами как «самая креативная» модель для текстов
- **Персональный контекст** — можно настроить Gemini под свою роль

## Пример: настройка персонального контекста

Gemini поддерживает персональные инструкции. Настройка «я CEO, не пишу код, даю промпты» меняет поведение: вместо фрагментов кода бот начинает давать промпты и стратегические рекомендации.
