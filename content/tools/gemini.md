## Обзор

Gemini — AI-ассистент от Google. Выделяется визуально богатыми интерфейсами по умолчанию и контекстным окном в 1M токенов у Gemini 2.5 Pro. Для работы с кодом слабее Claude Code и Cursor, но хорош для генерации контента и работы с большими документами.

## Для чего использовать

- **Генерация контента** — описан как "самая креативная" модель для текстов
- **Визуальные артефакты** — лучший фронтенд по умолчанию при генерации сайтов
- **Работа с документами** — контекст 1M токенов позволяет обрабатывать целые книги
- **Анализ стиля** — хорошо работает с экспортами текстов для извлечения тона и стиля

## Для кого

Для задач, где важна визуальная привлекательность результата, работа с большими документами или креативная генерация текстов.
