## Обзор

Gemini — AI-ассистент и семейство моделей от Google. Выделяется визуально богатыми интерфейсами по умолчанию и самым большим контекстным окном — до 2M токенов у Gemini 2.5 Pro. Для работы с кодом слабее Claude Code и Cursor, но хорош для генерации контента и работы с большими документами. Цены значительно ниже конкурентов.

## Модели

- **Gemini 2.5 Pro** — флагман, 2M контекст, SWE-bench 77.4%, отличный для кодинга
- **Gemini 2.5 Flash** — быстрая и дешёвая альтернатива, $0.15/1M input токенов

Бесплатный доступ через AI Studio.

## Для чего использовать

- **Генерация контента** — описан как "самая креативная" модель для текстов
- **Визуальные артефакты** — лучший фронтенд по умолчанию при генерации сайтов
- **Работа с документами** — контекст до 2M токенов позволяет обрабатывать целые книги
- **Анализ стиля** — хорошо работает с экспортами текстов для извлечения тона и стиля
- **Большие кодовые базы** — огромное контекстное окно позволяет загружать целые проекты

## Для кого

Для задач, где важна визуальная привлекательность результата, работа с большими документами, креативная генерация текстов или работа с большими кодовыми базами при ограниченном бюджете на API.
