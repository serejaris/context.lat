## Обзор

GitHub Copilot — AI-ассистент, который работает как плагин в вашей существующей IDE. Не нужно менять редактор — Copilot встраивается в VS Code, JetBrains, Neovim.

## Ключевые фишки

- **Agent mode**: автономное выполнение задач прямо в VS Code
- **Coding Agent**: полностью автономный агент, работающий через GitHub PR
- **Мульти-IDE**: работает везде, не привязан к одному редактору
- **GitHub**: нативная интеграция с Issues, PR, Actions

## Для кого

Для тех, кто не хочет менять IDE и уже активно использует GitHub.
