## Обзор

Replit — облачная IDE с AI Agent, который может создавать полноценные приложения. Всё работает в браузере: код, база данных, хостинг.

## Ключевые фишки

- AI Agent создаёт приложения по описанию
- Встроенная база данных, не нужно настраивать
- Моментальный деплой — приложение сразу доступно по URL
- Поддержка множества языков

## Для кого

Для начинающих и для быстрого прототипирования. Не нужно ничего устанавливать — всё в браузере.
