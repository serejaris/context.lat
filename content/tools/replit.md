## Обзор

Replit — облачная IDE с AI Agent, который может создавать полноценные приложения. Всё работает в браузере: код, база данных, хостинг. Среди AI-билдеров ([v0](/tools/v0/), [Lovable](/tools/lovable/), [Bolt](/tools/bolt/)) выделяется одной важной функцией: умеет генерировать мобильные приложения через Expo (React Native).

## Быстрый старт: мобильное приложение

1. Создать проект в режиме **Mobile App** (не веб — частая ошибка)
2. Вставить промпт из [метапромптинга](/concepts/metaprompting/) в чат Replit
3. Подождать 4-10 минут, пока AI Agent генерирует код
4. Получить QR-код → отсканировать в Expo Go → приложение на телефоне
5. Итерировать: описать баг текстом в чат → агент исправляет

## Синхронизация с GitHub

1. Tools and Files → Git → Sign In → авторизация [GitHub](/tools/github/)
2. Создать репозиторий с осмысленным именем
3. Sync Changes → код уходит в GitHub
4. Обратный путь: GitHub → Replit Import → вставить URL репозитория

## Сравнение с альтернативами

| Критерий | Replit | [v0](/tools/v0/) | [Bolt](/tools/bolt/) | [Lovable](/tools/lovable/) |
|----------|-------|------|---------|----------|
| Мобильные приложения | Да (Expo) | Нет | Нет | Нет |
| Деплой | Встроенный | Vercel | Netlify | Собственный |
| Стоимость | $0-20/мес | $0-20/мес | $0-20/мес | $0-20/мес |
| Сила | Mobile + full-stack | UI-компоненты | Full-stack web | Визуальный редактор |

## Когда выбрать Replit

- Идея требует **мобильного приложения** — единственный билдер с React Native
- Нужен **полный цикл** в одном инструменте: код + БД + деплой
- Не хотите устанавливать ничего локально

## Доступ из РФ

Из России вход через VPN (например, IP Нидерландов). Без VPN регистрация может блокироваться.
