## Обзор

v0 — AI-билдер от Vercel для быстрого создания веб-приложений. Специализируется на React + Tailwind CSS + shadcn/ui. По текстовому описанию генерирует полноценный интерфейс, который можно опубликовать в один клик.

Ключевая ценность: быстрая визуальная проверка идеи. Три минуты от промпта до работающего приложения в интернете.

## Быстрый старт

1. Открыть [v0.dev](https://v0.dev) и зарегистрироваться (через Gmail или GitHub, чтобы не требовали телефон)
2. Написать промпт:

```
Сделай мне веб-приложение — трекер подписок
```

3. V0 генерирует React-приложение с интерфейсом
4. Кнопка **«Починить»** исправляет ошибки без промпта
5. Нажать **Publish** → получить ссылку на серверах Vercel
6. При необходимости подключить [Supabase](/tools/supabase/) для базы данных — прямо из интерфейса V0

**Полезно:** функция «Enhance prompt» улучшает промпт перед генерацией.

## Примеры использования

### Трекер подписок (Урок 1)

На уроке создавался прямо во время занятия — 3 минуты, 0.8 кредита из 10 бесплатных. Один из студентов сделал трекер привычек для байкера ещё до конца урока.

### Прототип перед Cursor

V0 создаёт прототип → [Cursor](/tools/cursor/) / VS Code доводит до продакшена. V0 = визуальная проверка идеи, Cursor = доработка логики.

### Работа с Supabase

Подключение базы данных и авторизации прямо из интерфейса V0:
1. Создать проект в [Supabase](/tools/supabase/)
2. В V0: кнопка подключения → вставить ключи
3. V0 автоматически настраивает схему таблиц

## Ограничения

- **Бесплатный тариф**: 7 сообщений в день, 10 кредитов при регистрации
- **Кастомный домен**: недоступен на бесплатном тарифе — Vercel размещает на своём домене
- **Только веб**: мобильные приложения не генерирует
- **Фронтенд-фокус**: для бэкенда нужен Supabase или другой сервис
