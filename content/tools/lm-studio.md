## Обзор

LM Studio — приложение для запуска LLM локально на своём компьютере. Без облака, без подписок, без интернета. Модели скачиваются и работают полностью на устройстве.

## Для чего использовать

- Наглядная демонстрация работы LLM: видно, как модель генерирует токены
- Демонстрация среза знаний: локальная модель не знает событий после даты обучения
- Работа без интернета
- Эксперименты с разными моделями бесплатно

## Ограничения

Локальные модели значительно медленнее облачных: ~13 токенов/сек у локального DeepSeek против ~80 у облачного Claude. Качество тоже ниже — это компромисс между приватностью и производительностью.

## Для кого

Для экспериментов, обучения и случаев, когда нужна полная приватность данных.
