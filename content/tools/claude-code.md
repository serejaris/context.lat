## Обзор

Claude Code — это агентный инструмент для кодинга, который работает прямо в терминале. Не нужна IDE — Claude Code сам читает файлы, пишет код, запускает команды и тесты.

## Как работает

Запускаете `claude` в терминале, описываете задачу — Claude Code автономно анализирует проект, вносит изменения в нужные файлы, запускает тесты и отлаживает. Полный цикл разработки без переключения контекста.

## Ключевые фишки

- **Агентность**: не просто генерация кода, а полноценная работа с проектом
- **MCP**: подключение внешних инструментов через Model Context Protocol
- **Skills**: кастомные навыки через markdown-файлы
- **Hooks**: автоматизация через shell-команды на события

## Для кого

Для опытных разработчиков, которые живут в терминале и хотят максимальный контроль.
