## Обзор

Claude Code — агентный инструмент для кодинга, который работает прямо в терминале. Не нужна IDE — Claude Code сам читает файлы, пишет код, запускает команды и тесты. Агент видит весь проект, создаёт репозитории, деплоит на Railway, читает логи и чинит ошибки — полный цикл разработки без переключения контекста.

Также доступен через Desktop App: чат + вкладка для кода + режим планирования.

## Быстрый старт

1. Установить Claude Code:
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

2. Перейти в папку проекта и запустить:
   ```bash
   cd my-project
   claude
   ```

3. Описать задачу — агент автономно анализирует проект, вносит изменения, запускает тесты.

Пример первой задачи:

```
Создай Telegram-бота, который отвечает на сообщения.
Используй python-telegram-bot. Токен бота: BOT_TOKEN_HERE
```

Агент сам создаст файлы, установит зависимости и подготовит проект к запуску.

## Ключевые возможности

- **Агентность**: не просто генерация кода, а полноценная работа с проектом — чтение файлов, запуск команд, git-операции, деплой
- **[MCP](/mcp/)**: подключение внешних инструментов через Model Context Protocol. [EXA](/mcp/exa/) для поиска, [Chrome DevTools](/mcp/playwright/) для тестирования. Рекомендуется не более двух MCP одновременно — каждый занимает ~5% контекстного окна
- **[Skills](/concepts/skill/)**: переиспользуемые команды в файлах `.claude/skills/`. По сути — программирование агента на естественном языке
- **[Hooks](/concepts/hooks/)**: автоматизация через shell-команды на события — гарантирует выполнение важных действий детерминированно
- **[CLAUDE.md](/concepts/claude-md/)**: файл памяти проекта, который агент читает при каждом запуске — экономит токены
- **[Plan mode](/concepts/plan-mode/)**: режим планирования — агент анализирует задачу и предлагает план перед выполнением

## Примеры использования

### Деплой бота от идеи до облака

```
Создай Telegram-бота Second Brain Coach.
Используй OpenRouter API для LLM.
Задеплой на Railway.
```

Агент сам создаёт GitHub-репозиторий, пишет код, деплоит на Railway, а когда бот ломается — подключается к Railway CLI, читает логи и чинит.

### Контентный пайплайн

```
Создай пайплайн для публикации блог-постов:
1. Найди тему через EXA
2. Напиши черновик
3. Сгенерируй thumbnail
4. Опубликуй на сайт
```

Один skill — одна команда — полный цикл от идеи до публикации.

### Починка по логам

```
У меня ошибка на Railway, разберись
```

Агент сам запускает `railway logs`, находит ошибку, предлагает фикс и применяет его.

## Сравнение с альтернативами

| Критерий | Claude Code | [Cursor](/tools/cursor/) | [Codex](/tools/codex/) |
|----------|-------------|--------|-------|
| Интерфейс | Терминал / Desktop App | IDE (форк VS Code) | Веб (в ChatGPT) |
| Автономность | Высокая | Средняя | Высокая |
| Git-интеграция | Из коробки | Через IDE | Через GitHub |
| MCP/Skills | Да | Rules | Нет |
| Деплой | Через CLI | Нет | Через PR |
| Цена | $100-200/мес | $20/мес | $20/мес (Plus) |

**Связка:** Claude Code для планирования и архитектуры, Codex для пошаговой реализации готового плана.
