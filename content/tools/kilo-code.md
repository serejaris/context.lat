## Обзор

Kilo Code — open source плагин для VS Code, форк Cline. Подключается к любым моделям через свой API-ключ или Kilo Pass. Работает как бесплатная альтернатива Cursor с поддержкой новейших моделей через OpenRouter.

## Когда использовать

- Cursor упал или недоступен — Kilo Code как резервный вариант
- Нужен доступ к новым моделям (GLM 5.0, Kimi K2.5)
- Не готовы платить $20/мес за Cursor — Kilo Code бесплатен со своим ключом

## Для кого

Для тех, кто предпочитает VS Code и хочет бесплатную альтернативу Cursor с возможностью подключения любых моделей.
