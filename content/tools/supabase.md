## Обзор

Supabase — backend-as-a-service на PostgreSQL. Добавляет базу данных и авторизацию к приложениям в AI-билдерах (V0, Lovable, Bolt) без написания бэкенд-кода.

## Как подключить к V0

1. Создать проект на [supabase.com](https://supabase.com) -> получить URL и anon key
2. В V0: кнопка подключения Supabase -> вставить ключи
3. V0 автоматически настраивает схему таблиц
4. Авторизация через email/Google — в один клик

## Для чего использовать

Когда приложению нужна база данных или авторизация. Supabase берёт на себя бэкенд, позволяя сосредоточиться на продукте.

## Для кого

Для вайбкодеров, которые строят MVP в AI-билдерах и хотят добавить хранение данных без написания серверного кода.
