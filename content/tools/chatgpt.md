## Обзор

ChatGPT — чат-бот OpenAI, один из трёх основных AI-ассистентов наряду с Claude и Gemini. Для генерации веб-интерфейсов используется режим Canvas: нажать плюс, выбрать Canvas, вставить промпт — получить визуальный артефакт. Готовую работу можно расшарить через Share → Create Link.

Рекомендация: использовать 2-3 LLM параллельно. Каждая модель закрывает свою область лучше остальных. ChatGPT силён в работе со смыслами: формирование ТЗ, анализ рынка, генерация маркетинговых текстов.

## Быстрый старт

1. Открыть [chatgpt.com](https://chatgpt.com) (бесплатная регистрация)
2. Для генерации интерфейса: нажать **+** → выбрать **Canvas**
3. Вставить промпт:

```
Я хочу сделать трекер подписок. Задай мне пять вопросов
с тремя вариантами ответов по одному. С каждым вопросом давай
комментарии эксперта. В конце дай мне промпт на создание
приложения без указания технических деталей.
```

4. Ответить на вопросы → получить готовый промпт → вставить его в Canvas
5. Canvas сгенерирует интерфейс прямо в браузере

**Важно:** Canvas работает только в веб-версии, с телефона недоступен.

## Ключевые режимы

- **Чат** — стандартный диалог с моделью. Хорош для метапромптинга, PRD, анализа
- **Canvas** — совместное редактирование кода и интерфейсов в отдельном окне. Аналог Claude Artifacts, но только в браузере
- **[Codex](/tools/codex/)** — автономный агент для фоновых задач. Работает с GitHub, создаёт PR

## Примеры использования

### Метапромптинг для структурирования идеи

```
Я хочу сделать мобильное приложение для учёта расходов.
Задай мне 5 вопросов с 3 вариантами ответов, по одному.
В конце сформулируй PRD.
```

ChatGPT задаёт вопросы → вытаскивает контекст из головы → генерирует структурированный документ.

### Анализ комментариев

```
Проанализируй массив комментариев из TikTok-ролика [CSV-файл].
Составь портрет ЦА. Выдели JTBD. Напиши 3-5 гипотез для
продуктовых функций мобильного приложения.
```

### Сравнение моделей на одном промпте

Один и тот же промпт даёт разные результаты в ChatGPT, Claude и Gemini. Практическое упражнение: вставить метапромпт во все три — сравнить качество вопросов и финального артефакта.
