## Обзор

ChatGPT — чат-бот OpenAI, один из трёх основных AI-ассистентов наряду с Claude и Gemini. Для генерации веб-интерфейсов используется режим Canvas: нажать плюс, выбрать Canvas, вставить промпт — получить визуальный артефакт.

## Ключевые режимы

- **Чат** — стандартный диалог с моделью
- **Canvas** — совместное редактирование кода и текста в отдельном окне (только веб-версия)
- **Codex** — автономный агент для фоновых задач

## Для чего использовать

Хорош для работы со смыслами: формирование ТЗ, анализ рынка, генерация маркетинговых текстов, метапромптинг. Для глубокой работы с кодом — Claude Code или Cursor.

## Для кого

Универсальный AI-ассистент для повседневных задач. Рекомендуется использовать 2-3 LLM параллельно — каждая модель закрывает свою область лучше остальных.
