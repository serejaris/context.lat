## Обзор

Codex — агент OpenAI для написания кода и выполнения задач в фоновом режиме. Отличается медленной, но тщательной работой: проверяет каждый шаг, делает меньше ошибок. Входит в подписку ChatGPT Plus ($20/мес) — отдельной подписки не нужно.

Ключевое отличие от Claude Code: Codex работает в браузере, без установки и настройки. Описываете задачу в ChatGPT, Codex работает в фоне, результат приходит как Pull Request.

## Быстрый старт

1. Открыть [ChatGPT](https://chatgpt.com) с подпиской Plus или Pro
2. Перейти в раздел Codex (или описать задачу с пометкой «через Codex»)
3. Подключить GitHub-репозиторий
4. Описать задачу:

```
Вот PRD моего Telegram-бота (вставить PRD).
Реализуй его в репозитории my-bot.
```

5. Codex автономно анализирует репо, планирует изменения, пишет код и открывает Pull Request
6. Можно закрыть вкладку и вернуться к результату позже

## Ключевые возможности

- **Фоновая работа**: запустил задачу — закрыл вкладку — вернулся к готовому PR
- **Тщательность**: медленнее Claude Code, но реже ошибается — проверяет каждый шаг
- **Pull Requests**: результат оформляется как PR с описанием изменений
- **Без установки**: работает в браузере, не нужен терминал или IDE
- **Контекстное окно до 1M токенов**: может работать с большими кодовыми базами

## Примеры использования

### PRD → работающий бот

```
Реализуй Telegram-бота по этому PRD:
- Отвечает на сообщения в личке
- Использует OpenRouter API
- Хранит историю в SQLite
```

Codex создаст структуру проекта, напишет код и откроет PR.

### Рутинные задачи без присмотра

```
Добавь тесты для всех эндпоинтов в api/routes.py.
Покрытие минимум 80%.
```

Задачи, которые не требуют постоянного контроля — идеальный кейс для Codex.

## Связка с Claude Code

Хорошо работает как часть тандема:

```
Claude Code (план) → Codex (реализация)
```

1. В Claude Code: спланировать архитектуру, написать PRD, разбить на задачи
2. В Codex: передать PRD или отдельные задачи на реализацию
3. Codex выполняет в фоне, пока вы работаете над другими задачами
