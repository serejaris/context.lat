## Обзор

Codex — агент OpenAI для написания кода и выполнения задач в фоновом режиме. Отличается медленной, но тщательной работой: проверяет каждый шаг, делает меньше ошибок. Входит в подписку ChatGPT Plus.

## Как работает

Описываете задачу в ChatGPT -> Codex автономно анализирует репозиторий, планирует изменения, пишет код и открывает Pull Request. Работает в фоне — можно закрыть вкладку и вернуться к результату позже.

## Связка с Claude Code

Хорошо работает как часть тандема: Claude Code для планирования и архитектуры, Codex для пошаговой реализации готового плана.

## Для кого

Для тех, кто хочет автономного агента с минимальным контролем. Подходит для рутинных задач, которые не требуют постоянного присмотра.
