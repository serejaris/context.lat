## Обзор

Perplexity — AI-поисковик, который отвечает на вопросы с указанием источников. В отличие от [Claude](/models/claude/) или [ChatGPT](/tools/chatgpt/), Perplexity ищет актуальную информацию в интернете перед ответом. Каждый ответ — с ссылками на первоисточники.

## Быстрый старт

1. Открыть [perplexity.ai](https://perplexity.ai)
2. Задать вопрос с указанием даты для актуальности:

```
Топ библиотеки и технологии для Telegram-ботов на март 2026.
Сравни python-telegram-bot, aiogram и grammY.
Укажи текущие версии и активность.
```

3. Perplexity вернёт ответ с источниками
4. Результат сохранить как правило в конфигурации агента ([CLAUDE.md](/concepts/claude-md/))

## Примеры использования

**Ресёрч стека перед проектом:**
```
Лучшие инструменты для AI-автоматизации Telegram-каналов
на март 2026. Сравни подходы: API vs bot framework vs n8n.
```

**Проверка актуальности:**
```
Supabase pricing march 2026 — текущие тарифы и лимиты
free tier.
```

**Конкурентный анализ:**
```
Онлайн-курсы по vibe coding и AI-разработке в России 2026.
Кто проводит, формат, цены.
```

## Perplexity vs Claude vs Exa

| Критерий | Perplexity | [Claude](/models/claude/) | [Exa MCP](/mcp/exa/) |
|----------|-----------|--------------------------|---------------------|
| Интерфейс | Веб-чат | Чат-бот | MCP-сервер для агентов |
| Источники | Да, с ссылками | Нет (из памяти модели) | Да, сырой контент |
| Актуальность | Реальное время | До среза знаний | Реальное время |
| Глубина анализа | Средняя (обзор) | Глубокая (8 страниц) | Зависит от запроса |
| Использование | Ручной ресёрч | Структурированный анализ | Автоматический поиск агентом |

Claude даёт 8 страниц глубокого анализа, Perplexity — 35 страниц с источниками. Разные инструменты для разных задач.

## Рабочий процесс

1. **Perplexity** — быстрый ресёрч актуального стека
2. Результат → правило в [CLAUDE.md](/concepts/claude-md/) или `.cursor/rules`
3. Агент работает с проверенным стеком, не гадая по устаревшим знаниям

Важно: сохраняйте работающие решения и не меняйте стек без причины.
