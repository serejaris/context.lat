## Обзор

Perplexity — AI-поисковик, который отвечает на вопросы с указанием источников. В отличие от обычных LLM, Perplexity ищет актуальную информацию в интернете перед ответом.

## Для чего использовать

Быстрый ресёрч актуального стека перед началом проекта. Запрос с указанием даты ("Топ библиотеки для аналитики на март 2026") возвращает актуальные рекомендации с источниками. Результат сохраняется как правило в конфигурации агента.

## Perplexity vs Exa

| | Perplexity | Exa |

| Интерфейс | Веб-чат | MCP-сервер для агентов |
