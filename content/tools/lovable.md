## Обзор

Lovable — AI-билдер для создания полноценных приложений с визуальным редактором. Отличие от [v0](/tools/v0/) и [Bolt](/tools/bolt/): можно кликать по элементам интерфейса и редактировать их напрямую, не только через чат. Тесная интеграция с [Supabase](/tools/supabase/) для бэкенда.

## Быстрый старт

1. Открыть [lovable.dev](https://lovable.dev)
2. Описать приложение:

```
Создай CRM для фрилансера: список клиентов,
история проектов, трекер оплат. Авторизация через email.
```

3. Lovable генерирует приложение с интерфейсом
4. Кликнуть на любой элемент → отредактировать визуально или через промпт
5. Подключить [Supabase](/tools/supabase/) для базы данных и авторизации
6. Деплой на свой домен или домен Lovable

## Ключевые возможности

- **Визуальный редактор**: кликай и меняй — не только чат
- **[Supabase](/tools/supabase/) интеграция**: база данных и авторизация в один клик
- **GitHub sync**: код синхронизируется с GitHub для доработки
- **Кастомный домен**: можно привязать свой домен
- **Real-time preview**: изменения видны мгновенно

## Сравнение с альтернативами

| Критерий | Lovable | [v0](/tools/v0/) | [Bolt](/tools/bolt/) |
|----------|---------|----|----|
| Визуальный редактор | Да | Нет | Нет |
| Бэкенд | Supabase | Supabase | Встроенный |
| GitHub sync | Да | Нет | Экспорт |
| Кастомный домен | Да | Только Premium | Через Netlify |
| Сила | Визуальный контроль | Красивый UI | Мульти-стек |

## Когда выбрать Lovable

- Важен визуальный контроль над интерфейсом
- Нужна глубокая интеграция с Supabase
- Хотите синхронизацию с GitHub
- Нужен кастомный домен
