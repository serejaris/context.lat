## Обзор

Lovable — платформа для создания полноценных приложений через AI. Отличается от конкурентов визуальным редактором: можно кликать по элементам и редактировать их.

## Ключевые фишки

- Визуальный режим редактирования — кликаешь и меняешь
- Полная интеграция с Supabase для бэкенда
- Синхронизация с GitHub
- Деплой на свой домен

Для тех, кому важен визуальный контроль над интерфейсом и не хочется работать только через чат.
