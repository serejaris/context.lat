## Обзор

OpenAI предлагает линейку моделей для разных задач: GPT-4o для повседневной работы, o3 для сложного reasoning, и [Codex](/tools/codex/) для автономного программирования. Самая широкая экосистема: тысячи GPTs, интеграции с большинством инструментов, бесплатный доступ к базовым возможностям.

## Модели

| Модель | Назначение | Цена API (input) | Сила |
|--------|-----------|-------------------|------|
| **GPT-4o** | Мультимодальная, быстрая | $2.50/1M | Лучшая для работы с изображениями |
| **o3** | Сложные рассуждения | $10/1M | Reasoning, математика, логика |
| **Codex** | Автономный кодинг | через Plus/Pro | Агент в sandbox, результат = PR |

## Быстрый старт: Canvas

[ChatGPT](/tools/chatgpt/) Canvas — режим совместной работы над кодом:

1. Открыть [chatgpt.com](https://chatgpt.com), выбрать GPT-4o
2. Описать задачу — модель откроет Canvas с редактируемым кодом
3. Выделить фрагмент — попросить доработать конкретный участок
4. Canvas работает с кодом и текстами

Canvas доступен с телефона. Claude Artifacts тоже работает с мобильного — выбирайте по задаче.

## Codex: автономный агент

[Codex](/tools/codex/) — облачный агент OpenAI. Работает из [ChatGPT](/tools/chatgpt/) или API:

1. Описать задачу (или привязать к GitHub Issue)
2. Codex анализирует кодовую базу в sandbox
3. Пишет реализацию, запускает тесты
4. Результат — Pull Request в GitHub

Codex работает фоном — не занимает машину, пока вы работаете с другими агентами.
