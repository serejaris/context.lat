## Обзор

DeepSeek — китайская компания, создавшая одни из лучших open-source моделей. DeepSeek V3 и R1 показывают результаты на уровне проприетарных моделей при значительно меньших затратах.

## Модели

- **DeepSeek V3** — основная модель, 671B параметров, MoE архитектура
- **DeepSeek R1** — модель с продвинутым reasoning, конкурирует с o3

## Преимущества

Полностью открытый код — можно запустить локально или использовать через API. Цены на API самые низкие среди топовых моделей. R1 показывает впечатляющий reasoning.

## Для кого

Для тех, кто ценит open-source, хочет контролировать данные или оптимизировать расходы на API.
