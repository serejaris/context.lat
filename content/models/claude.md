## Почему Claude лучший для кодинга

Claude Opus 4.6 — флагманская модель Anthropic и безусловный лидер в генерации кода. На бенчмарке SWE-bench verified набирает 80.9% — это лучший результат среди всех моделей. На первом уроке Потока #10 Opus 4.6 вышел прямо во время эфира — студенты видели релиз в реальном времени.

## Модели семейства

| Модель | Назначение | Цена API (input) | Когда использовать |
|--------|-----------|-------------------|-------------------|
| **Opus 4.6** | Самая мощная, сложные задачи | $15/1M | Архитектура, планирование, оркестрация |
| **Sonnet 4.5** | Баланс скорости и качества | $3/1M | Основная рабочая модель для кодинга |
| **Haiku 4.5** | Быстрая и дешёвая | $0.25/1M | Рутинные задачи, парсинг, субагенты |

Haiku дешевле Opus в 60 раз — используйте для черновой работы и субагентов.

## Пример: метапромпт через Claude

```
Я хочу сгенерировать [сайт-визитку].
Задай мне 5 уточняющих вопросов, прежде чем
генерировать финальный промпт для AI-билдера.
```

Claude задаёт вопросы, уточняет контекст, генерирует структурированный промпт — который затем передаётся в [v0](/tools/v0/) или [Bolt](/tools/bolt/) для генерации кода.

## Claude vs ChatGPT для ресёрча

Claude даёт 8 страниц анализа ниши против 35 у [Perplexity](/tools/perplexity/) — разные инструменты для разных задач. Claude лучше для структурированного анализа, Perplexity — для объёмного ресёрча с источниками.
