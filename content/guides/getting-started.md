## Что такое вайбкодинг

Ты описываешь задачу словами — AI пишет код. Не нужно знать языки программирования, синтаксис или алгоритмы. Нужно уметь объяснить, что ты хочешь получить.

Вайбкодинг — это панк-рок программирования. Легендарный продюсер Рик Рубин не играет ни на одном инструменте, но создаёт хиты. Вайбкодер не пишет код руками, но создаёт работающие продукты — через промпты и управление [контекстом](/concepts/context/).

Под капотом работают [большие языковые модели (LLM)](/concepts/llm/) — они предсказывают следующий фрагмент текста на основе того, что ты им дал. Чем точнее контекст — тем лучше результат. А чтобы вытащить контекст из головы, есть техника [метапромтинга](/concepts/metaprompting/) — когда AI задаёт тебе уточняющие вопросы вместо того, чтобы сразу отвечать.

## Выбери инструмент

| Если ты... | Начни с |
|------------|---------|
| Хочешь визуальный редактор, привык к VS Code | [Cursor](/tools/cursor/) |
| Хочешь работать в терминале, максимальный контроль | [Claude Code](/tools/claude-code/) |
| Хочешь результат без установки, всё в браузере | [Bolt](/tools/bolt/) или [Lovable](/tools/lovable/) |
| Хочешь быстро собрать UI-компонент или веб-приложение | [v0](/tools/v0/) |

**Cursor** — AI-first IDE с визуальным интерфейсом. Привычный редактор кода, но с AI-ассистентом, который понимает весь проект. Подходит тем, кто хочет видеть файлы и структуру.

**Claude Code** — агентный CLI, который работает в терминале. Автономно читает, пишет и запускает код. Подходит тем, кто хочет делегировать задачу целиком и получить результат.

**Bolt, Lovable, v0** — браузерные AI-билдеры. Ничего не устанавливать: открыл, описал задачу, получил работающее приложение с публичной ссылкой. Идеально для первого проекта.

## Установи

### Cursor

1. Скачай с [cursor.com](https://www.cursor.com/)
2. Открой папку проекта
3. Начни диалог с AI через `Cmd+L` (Mac) или `Ctrl+L` (Windows/Linux)

### Claude Code

1. Установи Node.js 18+ (если ещё нет)
2. В терминале:

```bash
npm install -g @anthropic-ai/claude-code
```

3. Перейди в папку проекта и запусти:

```bash
claude
```

### Bolt / Lovable / v0

Ничего не устанавливать. Открой в браузере:
- [bolt.new](https://bolt.new)
- [lovable.dev](https://lovable.dev)
- [v0.dev](https://v0.dev)

## Первый проект за 15 минут

### Шаг 1: Придумай конкретную задачу

Не «хочу выучить программирование», а «хочу сделать сайт про себя» или «хочу трекер подписок». Мотивация через личную проблему работает лучше абстрактного обучения.

### Шаг 2: Используй метапромпт

Вместо того чтобы сразу писать «сделай мне сайт», попроси AI помочь сформулировать задачу. Вставь в любой чат-бот:

```
Я хочу сделать про себя сайт.
Задай мне три вопроса с тремя вариантами ответов, по одному.
С каждым вопросом давай комментарии экспертов, которые могли бы это обсудить.
Вопросы задавай строго по одному.
В конце дай мне структурированный промпт на создание одностраничного сайта без указания технологий.
```

AI задаст уточняющие вопросы и на выходе даст структурированный промпт — гораздо точнее того, что ты написал бы сам. Это и есть [метапромтинг](/concepts/metaprompting/).

### Шаг 3: Сгенерируй и итерируй

Возьми полученный промпт и вставь в [v0](https://v0.dev), [Claude](https://claude.ai), [ChatGPT](https://chatgpt.com) или [Gemini](https://gemini.google.com). Посмотри результат. Не нравится — не начинай заново. Укажи конкретно, что изменить: «заголовок крупнее», «фон тёмный», «добавь секцию с проектами».

### Шаг 4: Опубликуй

В v0 нажми **Publish** — получишь публичную ссылку. В Bolt и Lovable деплой встроен. Для Cursor / Claude Code используй [Vercel](https://vercel.com) — бесплатный хостинг с деплоем в один клик.

Готово. У тебя есть работающий проект в интернете.
