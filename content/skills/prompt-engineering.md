## Что это

Промпт-инжиниринг — умение формулировать запросы к AI так, чтобы получать нужный результат. Это фундаментальный навык для любого вайбкодера.

## Ключевые техники

### 1. Будь конкретным
Плохо: "Сделай форму"
Хорошо: "Создай форму регистрации с полями email, пароль, подтверждение пароля. Валидация на клиенте. React + Zod."

### 2. Давай контекст
Опиши стек проекта, существующие паттерны, ограничения. Чем больше контекста — тем лучше результат.

### 3. Используй примеры
Покажи пример желаемого результата. Формат ввода → формат вывода.

### 4. Итерируй
Первый результат редко идеален. Уточняй, корректируй, дополняй.

### 5. Chain of Thought
Для сложных задач попроси модель рассуждать пошагово: "Подумай шаг за шагом".

## Частые ошибки

- Слишком общие запросы
- Отсутствие контекста проекта
- Попытка сделать всё за один промпт
- Не читать то, что модель сгенерировала
