## Что это

AI pair programming (вайбкодинг) — методология разработки, где AI выступает вашим напарником. Вы задаёте направление, AI пишет код, вы ревьюите и корректируете.

## Принципы вайбкодинга

### 1. Вы — архитектор, AI — исполнитель
Принимайте архитектурные решения сами. Делегируйте рутину AI.

### 2. Маленькие итерации
Не просите написать всё приложение за раз. Двигайтесь маленькими шагами: фича → тест → рефакторинг.

### 3. Всегда ревьюйте код
AI может генерировать код с багами, уязвимостями, неоптимальными решениями. Ваша задача — ревьюить каждый diff.

### 4. Используйте правильный инструмент
- Простая задача → Copilot autocomplete
- Фича на несколько файлов → Cursor Composer / Claude Code
- Прототип → Bolt / Lovable
- UI компоненты → v0.dev

## Workflow

1. Опишите задачу в промпте
2. AI генерирует решение
3. Ревьюйте diff
4. Корректируйте через follow-up промпты
5. Тестируйте
6. Коммитьте
